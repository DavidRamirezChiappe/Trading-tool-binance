# Binance Trading Tools

Herramienta de análisis técnico para Binance enfocada en:

-   Modo mercado (watchlist)
-   Modo posición (gestión de trades)

------------------------------------------------------------------------

## 🆕 Versión 4.1.3

### Estructura de carpetas

Snapshots/ ├── Historial/ ├── Mercado/ └── Posicion/

### Outputs

#### Mercado

-   1_Watchlist_YYYYMMDD_HHMMSS.txt
-   1_Watchlist_YYYYMMDD_HHMMSS.json

#### Posición

-   1\_`<SYMBOL>`{=html}\_summary.txt
-   1\_`<SYMBOL>`{=html}\_summary.json

------------------------------------------------------------------------

## Objetivo

Herramienta de apoyo a decisión, no ejecución automática.
